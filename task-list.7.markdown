task-list(7) -- list task(s) for bake(1)
=============================================

## SYNOPSIS

List task(s) for bake(1).

## DESCRIPTION

Lists available tasks.

## REQUIRE

In your tasks(7) file `require` the `list` task(s) using:

	require 'tasks/list';

## USAGE

	bake list

## BUGS

**task-list** is written in bash and depends upon `bash` >= 4.
